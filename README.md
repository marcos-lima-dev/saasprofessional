FinanceHub - Business Intelligence Dashboard
O FinanceHub é um ecossistema de gestão financeira de alta performance, projetado para transformar dados brutos em inteligência estratégica. Desenvolvido com Next.js 15, o dashboard oferece uma experiência fluida para controle de faturamento, análise de saúde do negócio e geração de relatórios executivos.

🛠️ Stack Tecnológica de Elite
Core: Next.js 15+ (App Router & Turbopack)

Engine de PDF: html-to-image & jsPDF

UI/UX: Tailwind CSS & Shadcn/UI

Gráficos: Recharts (Data Visualization)

Gestão de Estado: React Context API

Feedback: Sonner (Toasts determinísticos)

✨ Funcionalidades Implementadas
✅ PDF Engine Profissional: Exportação de relatórios em formato A4 Paisagem, com tratamento de cores modernas e layout adaptativo (Desktop/Mobile).

✅ Business Health Intelligence: Algoritmo dinâmico que calcula a saúde do negócio (Atenção/Saudável/Excelente) com base em Volume de Vendas e Ticket Médio.

✅ Dashboard Responsivo: Interface inteligente que adapta elementos complexos (como botões e tabelas) para dispositivos móveis sem perda de funcionalidade.

✅ Gestão em Tempo Real: Fluxo completo de pedidos com atualização instantânea de métricas e gráficos de faturamento.

🚀 Como Rodar o Projeto
Clone o repositório:

Bash
git clone https://github.com/marcoslima/financehub.git
Instale as dependências:

Bash
npm install
Inicie o desenvolvimento:

Bash
npm run dev
